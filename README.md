# Tillikum core module

Installation
-------------

Installed via [Composer](http://getcomposer.org/).

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/tillikum/tillikum-core-module"
        }
    ],
    "require": {
        "tillikum/tillikum-core-module": "dev-master"
    }
}
```
